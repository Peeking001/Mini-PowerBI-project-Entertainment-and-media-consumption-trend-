# Mini-PowerBI-project-Entertainment-and-media-consumption-trend:-
🚀 Power BI Mini Project | Entertainment & Media Consumption Trends Dashboard
I’m excited to share my latest Power BI project where I analysed Entertainment & Media Consumption Trends among Gen Z and Millennials using survey data collected via Google Forms.
📊 Total Respondents: 17
 📈 Tool Used: Power BI Desktop
 📌 Dashboard Type: Interactive & Filter-Enabled (Month-wise slicer included)
🔎 Key Insights from the Analysis:
• 70%+ respondents prefer online streaming over traditional television
 • Majority consume digital entertainment 1–2 days per week, while a significant group engages daily
 • YouTube and Netflix emerged as top preferred platforms
 • Mobile is the dominant device for content consumption
 • Comedy was the most preferred genre (41% of responses)
 • Short videos and movies remain equally popular content formats
 • Most users watch 10–30 short videos per session
 • Gaming engagement is relatively low, with most users spending 0–3 hours per day
